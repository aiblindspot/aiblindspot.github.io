Metrics Selection

An algorithm’s performance depends on the metric selected for evaluation. It is important to choose metrics that capture the desired outcome. For example, when scanning for cancer, we want to minimize false negatives, but in spam filtering we want to minimize false positives.

Have you considered?

While it will not be possible to select the ‘perfect’ metric, it will be helpful to monitor a set of metrics that quantify the models’ performance across several dimensions. The practitioner will then have to make sure to balance the various metrics during the training process.

Case study

ABC Widgets is building a candidate sourcing system and has overarching goals of diversity and long term employee retention. While most standard machine learning methods rely on evaluation metrics such as area under the ROC curve or accuracy, it is important for the implementation team to work closely with stakeholders to understand how to encode meaningful business metrics. A machine learning system might need a custom metric that not only maximizes accuracy, but also captures the diversity of the predictions. These custom metrics depend on domain expertise and application awareness.
